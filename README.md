# Webbly

A modern, animated web application built with Next.js and React, featuring smooth animations and a beautiful UI.

**Live Demo:** https://webbly-next.vercel.app

## Overview

Webbly.Next is a contemporary web application that combines the power of Next.js with engaging animations and responsive design. It leverages modern React patterns and Tailwind CSS for a polished user experience.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) 15.3.8
- **UI Library:** [React](https://react.dev/) 19.0.0
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) 3.4.17
- **Animations:** [Framer Motion](https://www.framer.com/motion/) 11.16.4
- **Icons:** [Lucide React](https://lucide.dev/) 0.475.0
- **Utilities:** 
  - [date-fns](https://date-fns.org/) - Date manipulation
  - [clsx](https://github.com/lukeed/clsx) - Conditional CSS classes
  - [class-variance-authority](https://cva.style/) - Type-safe component variants
- **Language:** JavaScript (96.6%), TypeScript (2.6%), CSS (0.8%)

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/anishalvadzic/webbly.next.git
cd webbly.next
```

2. Install dependencies:
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the application for production
- `npm start` - Start the production server
- `npm run lint` - Run ESLint to check code quality
